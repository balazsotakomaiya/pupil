Front
What is the vanishing gradient problem?
Back
When gradients become extremely small as they're propagated backward through many layers, earlier layers learn very slowly or stop learning. Common with sigmoid/tanh. Mitigated by ReLU, residual connections, batch normalization.

Back

What is a transformer's self-attention mechanism?
Back
Self-attention computes a weighted sum of all positions in a sequence, where each weight reflects how relevant one token is to another. Uses query, key, and value projections with scaled dot-product attention: softmax(QK^T / √d_k) × V.
