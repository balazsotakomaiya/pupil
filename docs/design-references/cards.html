What is the vanishing gradient problem?
Back
When gradients become extremely small as they're propagated backward through many layers, earlier layers learn very slowly or stop learning entirely. Common in deep networks using sigmoid or tanh activations. Mitigated by ReLU, residual connections, or batch normalization.

Explain the difference between ownership and borrowing in Rust
Back
Ownership means a value has exactly one owner; when the owner goes out of scope, the value is dropped. Borrowing creates a reference (&T or &mut T) that lets you use a value without taking ownership. Borrows are checked at compile time: multiple shared borrows OR one mutable borrow, never both.

Back

What causes a move in Rust versus a copy? Rust Ownership New Manual —
Front
What causes a move in Rust versus a copy?
Back
Types that implement the Copy trait (integers, floats, bools, char, tuples of Copy types) are copied on assignment. All other types are moved — the original binding becomes invalid. Heap-allocated types like String, Vec, and Box are moved because copying their data would be expensive and implicit.
